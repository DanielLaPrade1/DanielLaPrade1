## Featured Projects

### 🌱 [ProjectSelfSufficiency](https://github.com/DanielLaPrade1/ProjectSelfSufficiency)
> **A full-stack food independence modeling application**

Combines agricultural data with a scalable backend to help gardeners, homesteaders, and small-scale producers understand and improve their level of food self-sufficiency. Users input their personal attributes and crops grown; the system returns a caloric self-sufficiency score, nutritional gap analysis, and actionable crop recommendations.

**Example output:**
```
"You are 22% self-sufficient based on estimated yearly caloric production."
"Key Insights: High in Carbohydrates, Low in Protein"
"Crop Recommendations: Lentils, Carrots, Soybeans"
```

**Questions the app helps answer:**
- *Can I sustain myself with what I currently grow?*
- *Which crops contribute the most nutritional value?*
- *What nutrient or caloric gaps exist in my system?*

### 🃏 [PokerHome](https://github.com/DanielLaPrade1/PokerHome)
> **Club management software for serious poker players**

An all-in-one platform for poker clubs to manage players, games, tournaments, and performance statistics. Built with a focus on real-world usability for poker clubs.

**Highlights:**

### 🏀 [BasketballPredictionModels](https://github.com/DanielLaPrade1/Basketball-Model)
> **ML models for predicting college basketball game outcomes**

Trained on scraped data from sports-reference.com, this project explores multiple modeling approaches to predict home/away team scores for NCAA basketball games. Includes both a neural network and a decision tree implementation, with an achieved MSE of ~95.
